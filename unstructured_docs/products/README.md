# Fiches Produits — Catalogue Éditorial

Ce dossier contient les fiches produits détaillées pour une sélection représentative du catalogue. Chaque fiche inclut les informations commerciales, la description, le public cible et l'historique de performance.

## Organisation

Les produits couvrent les 3 verticales et 6 catégories du catalogue :

| Vertical | Catégories | Fiches |
|----------|-----------|--------|
| **Littérature** | Roman, Essai | Les Ombres du Temps, Le Dernier Voyage, Le Jardin Secret, Histoires Fantastiques, La Société Numérique |
| **Jeunesse** | Jeunesse, Bande Dessinée | Éclats de Lune, La Maison Bleue, Super-Héros Academy |
| **Éducation** | Manuel Scolaire, Universitaire | Le Silence des Forêts, Sous le Ciel Étoilé, Manuel de Français CE1, Physique Quantique |

## Correspondance avec les données

Les fiches sont corrélées avec :
- `product_dim.csv` — Référence produit, catégorie, vertical
- `sales_fact.csv` — Historique des ventes
- `product_category_dim.csv` — Classification par catégorie et vertical
